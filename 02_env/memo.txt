次のコマンドを実行してDockerImageを作成する。
docker build -t bill_image .

次のコマンドを実行してコンテナの作成と起動を行う。
docker run -d --name bill_container -p 5432:5432 -e POSTGRES_USER=myuser -e POSTGRES_PASSWORD=mypassword -e POSTGRES_DB=mydatabase bill_image

コンテナにアクセスしてDBServerにアクセスする。
docker exec -it bill_container psql -U myuser -d mydatabase
